# 🌦 Weather Application using OpenWeatherMap API

## 📌 Objective
The objective of this project is to build a dynamic weather application that fetches **real-time weather data** using the OpenWeatherMap API and displays it on a user-friendly interface.  
Students will apply concepts learned in **HTML, CSS, JavaScript, and API integration** to create a functional web application.

---

## 📝 Project Description
This Weather Application allows users to:
- Search for a city
- View its **current weather details**
- See a **5-day forecast**
- Experience a responsive and visually engaging interface with dynamic backgrounds

---

## ✅ Features
### Minimum Requirements
- 🔍 **City Search**: Enter a city name to fetch weather information.
- 🌡 **Weather Information Display**:
  - City Name
  - Temperature
  - Weather Condition (Cloudy, Sunny, Rainy, etc.)
  - Humidity
  - Wind Speed
- 🔗 **API Integration**: Uses OpenWeatherMap API for real-time data.
- 🎨 **User Interface**: Clean and responsive design with HTML & CSS.
- ⚠️ **Error Handling**: Displays a proper message if the city is not found.

### Enhancements
- 🌤 **Weather Icons** (from OpenWeatherMap)
- 📅 **5-Day Weather Forecast**
- 🎨 **Dynamic Backgrounds** based on weather condition
- ⏳ **Loading Animations**
- 📱 **Mobile Responsive Design**

---

## 🛠️ Technologies Used
